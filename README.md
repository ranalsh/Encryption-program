# Encryption/Decryption System

Java program that manages 3 keys and 2 sentences (or more) using a menu system.
Supports setting keys (with PIN), selecting a sentence object, entering a sentence, and encrypting/decrypting it using substitution mapping.

# Files

- JavaProject2.java – main menu + program flow

- Key.java – key validation & storage

- SecureSentence.java – sentence storage, encrypt/decrypt

Run
javac JavaProject2.java Key.java SecureSentence.java
java JavaProject2
